GZIP64 API

GZIP64 API is a simple online gzip service!
Just submit your token using the "api_token" parameter and your data using the "query" paramater to the
"/api/v1/compress"endpoint to get your compressed data.


We're currently in public beta, but we're planning on releasing special enterprise versions, contact us for more info!
